# Componentes de Rede: Hardware

> **Metodologia:** Kumon (Pequenos passos, Exemplos, Prática imediata)

---

## Passo 1: O Modem

**Explicação:**
É a "porta de entrada". Ele traduz o sinal que vem da rua (Fibra Óptica ou Cabo) para o computador entender.
**Função:** Conectar sua casa à Internet (Provedor).

### Exercício 1
Sem o modem, a internet da rua consegue chegar no seu PC?
( ) Sim
( ) Não

---

## Passo 2: O Roteador (Router)

**Explicação:**
É o "guarda de trânsito". Ele pega a internet do Modem e **divide** para vários dispositivos (Celular, TV, PC).
Geralmente, Modem e Roteador vêm juntos no mesmo aparelho hoje em dia (O "Wi-Fi").
Ele cria a rede **Wi-Fi**.

### Exercício 2
Se você tem internet no computador, mas o Wi-Fi do celular não conecta, o problema provável é no:
( ) Modem (Internet da rua)
( ) Roteador (Distribuição Wi-Fi)

---

## Passo 3: Cabos e Interfaces

- **Cabo de Rede (RJ-45):** Aquele cabo azul ou cinza. É mais rápido e estável que o Wi-Fi.
- **Placa de Rede:** A peça dentro do PC onde encaixa o cabo.

**Dica:** Para jogos e downloads pesados, prefira cabo!

### Exercício 3
Para ter a conexão mais rápida e estável possível no seu quarto, você usa:
( ) Wi-Fi
( ) Cabo de Rede
