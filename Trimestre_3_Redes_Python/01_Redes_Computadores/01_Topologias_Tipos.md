## Passo 1: O que é uma Rede?

**Explicação:**
Rede é quando conectamos dois ou mais computadores para compartilhar coisas (arquivos, internet, impressora).
- **LAN (Local Area Network):** Rede Local. Pequena (sua casa, uma sala de aula).
- **WAN (Wide Area Network):** Rede Grande. O mundo todo (A Internet).

### Exercício 1

( ) WAN (Mundial)

## Passo 2: Topologias (Desenhos da Rede)

**Explicação:**
É o "formato" como os cabos conectam os computadores.

1.  **Estrela (Star):** Todos ligados em um aparelho central (roteador). Se um cabo quebra, só aquele PC fica sem net. (É a mais usada hoje!).
2.  **Barramento (Bus):** Todos ligados em um único fio. Se o fio principal quebra, a rede toda para. (Antigo).
3.  **Anel (Ring):** Um liga no outro, formando um círculo.

## Passo 3: Wi-Fi (Sem Fio)

**Explicação:**
**WLAN** (Wireless LAN).
Usa ondas de rádio em vez de cabos.
Vantagem: Mobilidade.
Desvantagem: Pode ter interferência (paredes, micro-ondas).

### Exercício 3

( ) World (Mundo)
( ) Wireless (Sem fio)
