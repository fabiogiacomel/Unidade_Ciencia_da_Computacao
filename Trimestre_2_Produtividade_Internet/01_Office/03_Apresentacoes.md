# Apresentações (PowerPoint / Slides)

> **Metodologia:** Kumon (Pequenos passos, Exemplos, Prática imediata)

---

## Passo 1: O Slide

**Explicação:**
O Slide é cada "tela" da sua apresentação.
Não é uma folha de papel! Ninguém gosta de ler textos gigantes no telão.
**Regra de Ouro:** Pouco texto, Tópicos e Imagens.

**Exemplo Ruim:** Um texto de 10 linhas copiados da Wikipédia.
**Exemplo Bom:** Uma foto bonita e uma frase: "A tecnologia mudou o mundo."

### Exercício 1
O que deve ter num bom slide?
( ) Textões longos para mostrar que eu estudei.
( ) Imagens e tópicos curtos para guiar minha fala.

---

## Passo 2: Contraste (Cores)

**Explicação:**
As pessoas precisam ler o que está escrito.
- Fundo **Escuro** -> Letra **Clara**. (Fundo Preto / Letra Branca)
- Fundo **Claro** -> Letra **Escura**. (Fundo Branco / Letra Preta)

**Erro Comum:** Fundo Azul Escuro com Letra Vermelha (Dói o olho e ninguém lê).

### Exercício 2
Se o fundo do slide é **Preto**, qual cor de letra é melhor?
( ) Cinza escuro
( ) Branco ou Amarelo claro

---

## Passo 3: Transições e Animações

**Explicação:**
- **Transição:** É o efeito quando muda de um slide para o outro (ex: Deslizar).
- **Animação:** É o efeito nos itens do slide (ex: A foto aparece girando).

**Cuidado:** Excesso de animação cansa a plateia e parece infantil. Use com moderação!

### Exercício 3
É legal colocar som de explosão e efeitos giratórios em TODOS os slides de um trabalho sério?
( ) Sim, fica divertido.
( ) Não, tira a atenção do conteúdo.

## Exercício Prático no Computador

1.  Crie uma apresentação com 2 Slides.
2.  **Slide 1 (Capa):** Título "Meu Hobby Favorito" e seu Nome.
3.  **Slide 2:** Uma imagem do seu hobby e uma frase curta explicando por que gosta.
4.  Use contraste correto (Fundo x Letra).

**Checklist:**
[ ] Tem 2 slides.
[ ] Pouco texto no slide 2.
[ ] Dá para ler bem (boas cores).
