## Passo 1: Números Gigantes

**Explicação:**
Na computação e ciência, lidamos com números enormes (tamanho do universo) ou minúsculos (tamanho de um átomo).
Escrever `300.000.000` (velocidade da luz) dá trabalho.
Usamos **potências de 10** para simplificar.

**Regra:** Conte quantos zeros tem.
`100` = 10² (2 zeros)
`1000` = 10³ (3 zeros)
`1.000.000` = 10⁶ (6 zeros)

**Exemplo:**
`300.000.000` = 3 x 10⁸ (8 zeros).

### Exercício 1
Como escrever `5000` em notação científica?
( ) 5 x 10²
( ) 5 x 10³

## Passo 2: Números Minúsculos (Decimais)

**Explicação:**
Para números menores que 1 (0,00...), o expoente fica **negativo**.
Conte quantas casas a vírgula anda para a direita.

**Exemplo:**
`0,001` = 1 x 10⁻³ (vírgula andou 3 casas).
`0,000005` = 5 x 10⁻⁶.

### Exercício 2
Como escrever `0,02` em notação científica?
( ) 2 x 10⁻²
( ) 2 x 10²

## Passo 3: Aplicação na Computação (Hertz e Bytes)

**Explicação:**
Usamos isso para velocidade (Hz) e tamanho (Bytes).
- **Kilo (K):** 10³ (Mil)
- **Mega (M):** 10⁶ (Milhão)
- **Giga (G):** 10⁹ (Bilhão)

**Exemplo:**
Um processador de **3 GHz** (GigaHertz) faz **3 x 10⁹** (3 bilhões) de ciclos por segundo.

### Exercício 3
Se você tem um HD de **1 TB** (Tera), são 10¹² bytes.
Isso é:
( ) 1 Bilhão
( ) 1 Trilhão
