## Passo 1: Software de Sistema

**Explicação:**
São os programas essenciais para o computador ligar e funcionar. Sem eles, o hardware é apenas metal e plástico inútil.
O principal exemplo é o **Sistema Operacional**.

**Exemplos:**
- Windows 10/11
- Linux (Ubuntu, Fedora)
- Android / iOS
- drivers (programas que fazem a impressora ou webcam funcionar)

## Passo 2: Software de Aplicação (Aplicativos)

**Explicação:**
São os programas focados em **tarefas do usuário**. Você usa para criar coisas, jogar ou navegar.
Eles precisam do sistema operacional para rodar.

**Exemplos:**
- Navegadores (Chrome, Firefox)
- Editores (Word, Bloco de Notas)
- Jogos (Fortnite, Roblox)
- Mensageiros (WhatsApp, Discord)

## Passo 3: Software de Programação

**Explicação:**
São ferramentas usadas pelos programadores para **criar novos softwares**.
São como "fábricas de programas".

**Exemplos:**
- Python (Linguagem que vamos usar!)
- Visual Studio Code (Editor de código)
- Scratch (Programação em blocos)
